Key Instructions

This calculator adapts ‘PEDMAS’ as order of applying operator.

It means the order of applying operators will be parenthesis, exponents, multiplication|division, addition|subtraction. Any same level of operator will go from left to right.

For example, 8/2(2+2) will equals to 16 instead of 1.

Memory System

The memory system can be accessed by pressing M+ and M- keys, M- will undo your action to the last result produced and M+ will serve to revert the undos made previously.

"M" will display your memory cache in console
